Project proposal for The Data Incubator

Repo contains web scraper for drinksmixer.com, which pulled information on all the drinks and saved it in the hdf5 file. Initial analysis has been done producing two plots to support the project proposal.

Plot 1

Distribution of calories per 1 oz of drink for all drinks with nutritional information. .

Plot 2

Scatter plot of the drink ratings vs calorie per 1 oz for all drinks with nutritional information and more than 10 ratings. .
